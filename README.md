Customer geo-position provider and Yandex Map example module: module for PrestaShop 1.5-1.6
===========================================================================================

Description
------------
This is a module for [PrestaShop][4] e-commerce solution (v1.5-1.6) that works as data-provider and provides the example how to implement a geo-position feature on a map.
[Customer geo-position provider module for PrestaShop homepage][5].

Installation & Update
------------
The module has a normal installation.

To update the module just do this:
 - click uninstall button
 - remove all files from "/modules/customergeopos/"
 - copy new files to the same directory
 - and click install button
